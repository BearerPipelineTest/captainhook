Composer integration

If you provide a captainhook.json file in your project you can make sure CaptainHook will install the hooks to every developers local repository. Just add the following lines to your projects composer.json file. 

{
  "scripts": {
    "post-install-cmd": "\\SebastianFeldmann\\CaptainHook\\Composer\\Cmd::install"
  }
}

Including this lines, Composer will execute the CaptainHook install command every time composer install is executed.
