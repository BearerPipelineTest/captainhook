CaptainHook Documentation

CaptainHook is a flexible and easy to use git hook manager.

What's in it?

If you want to make sure nobody commits code that breaks you tests or coding standard, or you want to validate your commit messages, CaptainHook can help you to get going.

How does it work?

  • Install CaptainHook
  • Configure hooks
  • Write your own hooks
  • Composer integration

Requirements

The current version of CaptainHook (3.x) requires a minimum PHP version of 7.0.

If you are still using PHP 5.6 or earlier you have to use CaptainHook version 1.x.
